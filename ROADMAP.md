# 🗺️ Implementation Roadmap

This roadmap tracks the planned topics and implementation queue for the `quantum-playground` repository.

## Phase 1: Foundational Circuits & Core Simulator (Current Focus)
- [ ] **`circuits/basic`**: Bell states, GHZ states, quantum teleportation circuit (High Priority)
- [ ] **`simulators/statevector`**: NumPy-based statevector simulator (High Priority)
- [ ] **`circuits/arithmetic`**: Quantum half-adder and full-adder (Medium Priority)

## Phase 2: Quantum Algorithms & Noise Models
- [ ] **`algorithms/search`**: Grover's algorithm (2-qubit to n-qubit) (High Priority)
- [ ] **`noise_models`**: Bit-flip, phase-flip, depolarizing channels (Medium Priority)
- [ ] **`simulators/density_matrix`**: Density matrix simulator with noise (Medium Priority)

## Phase 3: Advanced Algorithms & Error Correction
- [ ] **`algorithms/factoring`**: Shor's algorithm (step-by-step) (High Priority)
- [ ] **`algorithms/simulation`**: VQE for H₂ molecule ground state (Medium Priority)
- [ ] **`error_correction`**: 3-qubit bit-flip code, Steane [[7,1,3]] code (Low Priority)
